HTML5 Image Example 1

This is an example using the HTML5 srcset and sizes attributes

The purpose is to provide the browser with the images we have and the sizes. Then we also tell the browser how wide the image should be displayed at any given viewport width. With this information the browser can decide which image is appropriate to use for the given size. Imagine the img tag as a container. We as developers only define how big the container should be at any given point and the browser decides which source to load to get the best output for the user.

100vw stands for 100% of the viewport width. You can also do 200vw for more high res images.

Currently this is in draft mode so it may change as time goes by

To test the srcset and sizes attributes using Chrome do the following

  1. Open Chromes inspector, click the "network" tab and check the "Disable Cache" checkbox
  2. View the page with the browser window greater than 953 pixels. You will notice the image does not fill the entire page.
  3. Keeping the same browser size refresh the page. Under the network tab you will see the largest picture "images/ambulance-953x520.jpg" as downloaded
  4. Move the browser window in so the page is less then 785 and refresh the page. Under the network tab you should see the medium picture loaded "images/ambulance-785x428.jpg"
  5. Move the browser window in so the page is less then 628 and refresh the page. Now the picture loaded will be the smaller one "images/ambulance-628x327.jpg"

Source code

HTML
<img srcset="images/ambulance-953x520.jpg 953w, images/ambulance-628x327.jpg 628w, images/ambulance-785x428.jpg 785w" sizes="(min-width: 953px) 953px, 100vw" alt="HART response vehicle">

CSS
#main {margin: 0 auto; max-width: 1000px;}
img {max-width: 100%;}
