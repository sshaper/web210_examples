Star Rating Example

This example uses the following aria attributes:

  • role="slider" - Sets the role to slider
  • aria-valuemin - The minimum value of the slider
  • aria-valuemax - The maximum value of the slider
  • aria-valuenow - The current value of the slider
  • aria-valuetext - Used to define the human readable text alternative of aria-valuenow for a range widget such as progressbar, spinbutton or slider.

This example uses both the aria-valuenow and the aria-valuetext attributes just to show them. Aria-valuetext is the alternative to aria-valuenow.

Tab to highlight the rating stars then use the up and down arrow keys to increase and decrease the stars. This is a keyboard only application the mouse will not work.
