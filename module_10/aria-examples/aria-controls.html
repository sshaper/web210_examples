How to use this application

This application was tested using Chromevox for Google chrome. For mouse users just click on the appropriate tab. For keyboard users use the tab key to select the appropriate tab and press the enter key. To get back to the menu press the tab key (note the tab order will start with the first one). For chromevox users use the tab key to select the appropriate tab then press chromevox keys+enter. NOTE: you must use the chromevox keys shift+altfor this to work. To get back to the menu press the tab key.

Press shift+tab to go backwards through the tab list.

Note: In Firefox for Mac you have to set your preferences (in Mac) to accept all controls. System preferences->keyboard->(check all controls).
